ETAPA 01
Oque são?
São operadores utilizados para comparar dois valores

IGUALDADE
saldo = 450
saque = 200

print(saldo == saque)
False (sempre booleano)

DIFERENÇA
saldo = 450
saque = 200

print(saldo != saque)
True

MAIOR QUE / MAIOR OU IGUAL
saldo = 450
saque = 200

print(saldo > saque)
True

print(saldo >= saque)
True

MENOR QUE / MENOR OU IGUAL
saldo = 450
saque = 200

print(saldo < saque)
False

print(saldo <= saque)
False
